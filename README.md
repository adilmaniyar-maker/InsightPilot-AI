# InsightPilot AI

AI-powered data analytics platform built with Streamlit, Pandas, Plotly, Scikit-Learn and Gemini AI.

## Features
- Data Cleaning
- KPI Dashboard
- Forecasting
- Anomaly Detection
- AI Business Insights
- Chat with Data
- Download Reports

## Tech Stack
Python, Streamlit, Pandas, Plotly, Scikit-Learn, Gemini AI
