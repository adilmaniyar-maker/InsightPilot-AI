# InsightPilot AI

An AI-powered Data Analytics Platform built using Python, Streamlit, Plotly, Scikit-Learn, and Gemini AI.

## Features
- Data Cleaning
- Interactive Dashboard
- Forecasting
- Anomaly Detection
- AI Business Insights
- Chat with Data
- Executive Summary
- PDF & Excel Reports
- Data Quality Score

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-Learn
- Gemini AI
- ReportLab
