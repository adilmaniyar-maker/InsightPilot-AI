# 🚀 InsightPilot AI - AI Powered Data Analytics Platform

An AI-powered Data Analytics Platform built using Python, Streamlit, Plotly, Scikit-Learn, and Google Gemini AI. This application enables users to upload datasets, perform automated data cleaning, visualize insights, generate AI-powered analysis, and build machine learning models with an intuitive interface.

---

## 📌 Features

- 📂 Upload CSV and Excel datasets
- 🧹 Automatic Data Cleaning
  - Handle Missing Values
  - Remove Duplicate Records
  - Data Type Detection
- 📊 Interactive Data Visualization
  - Bar Chart
  - Line Chart
  - Scatter Plot
  - Histogram
  - Box Plot
  - Correlation Heatmap
- 📈 Dataset Statistics
- 🤖 AI-powered Insights using Google Gemini
- 🔍 Exploratory Data Analysis (EDA)
- 📉 Machine Learning Predictions
- 📥 Download Processed Dataset
- ⚡ Fast and Interactive Streamlit Dashboard

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/adilmaniyar-maker/InsightPilot-AI.git
```

Go to project directory

```bash
cd InsightPilot-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
streamlit run app.py
```
